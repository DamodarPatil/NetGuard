## How to Submit

1. **Fork the repository**
   ```bash
   git clone https://github.com/[repo-owner]/[awesome-list].git
   cd [awesome-list]
   ```

2. **Create a new branch**
   ```bash
   git checkout -b add/flowsentrix
   ```

3. **Edit the README.md** — Find the appropriate section and add the entry (keep alphabetical order if required)

4. **Commit & Push**
   ```bash
   git add README.md
   git commit -m "Add FlowSentrix to [list-name]"
   git push origin add/flowsentrix
   ```

5. **Create a Pull Request** on GitHub with a clear description

## Tips for Success

✅ **Do:**
- Keep description clear and under 100 characters
- Explain what makes it unique or useful
- Ensure links are working
- Follow the list's existing format
- Check if it's already listed

❌ **Avoid:**
- Self-promotional language ("the best", "amazing")
- Spamming multiple lists at once
- Descriptions that are too long
- Submitting to lists where it doesn't fit
- Duplicate submissions
